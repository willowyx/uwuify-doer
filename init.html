Important note for Firefox users

Due to the current implementation of Manifest v3, Firefox doesn't allow the extension access to all sites by default, so you'll end up having to click it every time you want to activate it, which sort of defeats the purpose of having a perma-uwu mode.

Fortunately, there's a fix:

The easiest way is to right-click on the extension in the toolbar and select "Manage Extension". From there, select the "Permissions" tab and toggle "Access your data for all websites". This will allow the extension to run as intended. I'd like to note here that the extension collects no data, and is open-source if you wish to verify this!

If you do not feel comfortable doing this, you can also enable the extension on a site-by-site basis: navigate to the site of your choosing, right-click on the extension in the toolbar, and select "Always allow on [site name]". This will allow the extension to run on that site without requiring you to click it every time.
