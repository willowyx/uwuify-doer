Note for Firefox

If you're using Firefox:

If you're using an older version of Firefox, you may have to grant uwuify permissions due to Firefox's implementation of Manifest v3, an extension protocol.

When you press the button below, Firefox will prompt you to grant these permissions. You can enable this at any time, so no pressure.

Request permissions
