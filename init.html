Note for Firefox

If you're using Firefox:

uwuify uses Manifest v3, and Firefox's implementation of it requires that you (the user) manually grant it additional permissions, instead of having them enabled by default when you install it. uwuify requires the permissions "read and change your data for all sites" in order for most of its functionality to work (uwuifying text automatically, whitelisting sites, etc.)

If you're okay with this, press the button below and you will be prompted by Firefox to grant these permissions. You can still use the extension without granting these permissions but functionality will be very limited.

Request permissions
